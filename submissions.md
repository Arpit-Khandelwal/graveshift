# Graveyard Hack Submission

Submit once per project via Typeform:

- https://solanafoundation.typeform.com/graveyardhack

This single form covers:

- Main track submission
- Sponsor bounty applications

## Deadline

- Submissions close on **February 27 at 11:59 PM UTC**.
- Late submissions are not accepted.

## Required In Form

- Project name
- One-line description
- GitHub repo (must stay public until March 5)
- Demo/pitch video
- Optional relevant links (deployed app, artwork, X handle)
- Sponsors you are applying for
- What is next on your roadmap
- Longer project description (include how each sponsor was used)
- Telegram handle

## Submission Checklist

- [ ] Filled Typeform once for the project
- [ ] Added public GitHub repo link
- [ ] Added demo/pitch video link
- [ ] Selected target sponsors
- [ ] Added roadmap and longer description
- [ ] Included Telegram handle
- [ ] Submitted before deadline

## Notes

- Once the form is submitted, the project is considered submitted.
- Organizers will only contact you for follow-up questions or winner notifications.
